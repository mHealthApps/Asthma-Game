# asthma-research

Environment setup instructions:

Download Node.js
1. Go to https://nodejs.org/en/download/prebuilt-installer
2. Make sure OS is correct and click 'Download Node.js' button
3. Navigate the Node.js Setup Wizard, clicking next and agreeing to the terms, etc
4. Click Install and do not close the installer until it is complete (press "Finish" when done)
5. Verify Node.js was properly installed by going into command prompt and running 'node -v'

Update npm
1. Run 'npm install --global'

To run React app:
1. Clone repository to GitHub Desktop
2. cd into the repository in the command line
3. cd into asthma-app directory
4. run 'npm install'
5. run 'npm start' to run a developer build of the app at http://localhost:3000/
