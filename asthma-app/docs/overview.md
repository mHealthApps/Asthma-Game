# App Overview

## Purpose
The purpose of this application is to provide asthma education targeted to Native Hawaiian families.

## Features
The app contains 6 learning modules covering asthma-related topics.

Including:
- Interactive educational content
- Embedded PixiJS learning games
- Progress tracking


## Tech Stack
Frontend: 
- React
- PixiJS (for the games)

Backend:
- Flask
- SQLAlchemy

Database:
- PostgreSQL
