# PixiJS Games Overview

## Purpose
The purpose of the games is to retain the attention of a younger demographic and facilitate asthma learning in the process.

## Features
The app features 3 different types of games:
- Falling Objects Gate (demo-game/) --> used in HealthyLifestyle and FirstAid modules
- Gate Game (gate-game) --> used in TheLungs and AboutAsthma modules
- Memory Game (memory-game) --> used in AsthmaTreatment and AsthmaManagment modules

## Integration
Within the learning modules, a game is played at the end of a learning module.
Additionally the user has an option to turn off Game Mode and instead have quizzes at the end of a learning module.
